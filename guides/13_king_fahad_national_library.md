## 4. 블록아웃 (~30분)

### 추정 치수 (검증 전 출발점)

OSM·위성 측정 전 일반 추정:
- 본체 큐브: **~85 × 30 × 85m**
- 외피(막구조): 본체 밖 약 2-3m 간격으로 한 겹 감쌈 → ~91 × ~33 × ~91m
- 상단 파라펫: ~1-2m

### Phase A — 본체 + 외피 매스 (~300 tris)

**파라메트릭 유지**

```
KSA_13_King_Fahad_Library (Null)
├── body_cube      [Cube]  85 × 30 × 85m   @ P.Y=15      ← 본체 큐브
├── facade_wrap    [Cube]  91 × 32 × 91m   @ P.Y=16      ← 외피 셸 (속이 빈 매스)
│                  ※ 디테일 단계에서 마름모 패턴 적용
└── parapet_top    [Cube]  91 × 1.5 × 91m  @ P.Y=33.5    ← 상단 띠
```

**외피 분리 모델링 vs 단일 매스?**
- **분리** (권장): 본체와 외피를 별도 매스로 둠 → 외피 투명도/패턴 디테일 자유로움
- **단일**: 외피만 모델링하고 내부 무시 (간단하지만 안쪽 안 보임 — 디테일 한계)

→ 분리 권장. 내부 본체는 단순 cube, 외피만 패턴 디테일.

### Phase B — 마름모 패턴 외피 (~6,000–10,000 tris) ★ 시그니처

PTFE 막구조 외피가 이 건물의 정체성. 마름모(rhombus) 그리드가 외피 전체를 덮음.

**모델링 방식 옵션:**

**옵션 1 — 단일 마름모 + Cloner** (권장, 효율적)
- 마름모 1개 = 정사각형 회전 또는 Plane 4-vertex
- Cloner Grid 모드로 외피 4면(또는 5면 포함 지붕) 둘레 반복
- 1면당 가로 30 × 세로 12 = 360개 정도 → 4면 1,440개
- 1마름모 = 2 tris → ~2,880 tris

**옵션 2 — 텍스처로 처리**
- 외피 매스에 마름모 패턴 텍스처
- 단, 텍스처 미적용이 기본이라 비권장 (모델 자체에 디테일 필요)

**옵션 3 — 음각 컷아웃 (Boolean)**
- 외피 매스에서 마름모 모양 작은 큐브들을 빼냄
- 면당 변동성·디테일 풍부하지만 Tri 폭증 위험

→ **옵션 1 (Cloner) 권장**

**Tri 추산 (옵션 1):**
- 1마름모 (사각 plane) = 2 tris
- 면당 ~360개 × 4면 = 1,440개 → ~2,880 tris
- 지붕 포함 시 +1,000 tris

**체크포인트**: 약 3,500–6,000 tris

### Phase C — 진입부 디테일 (~500 tris)

- 진입 계단 / 광장으로 통하는 부분
- 본체 또는 외피의 일부 면이 열려 있는 진입 동선
- 단순 매스 추가 (Cube)

### Phase D — 검토 · 클린업

- [ ] Object Manager 모두 파라메트릭

- [ ] 비가시 면 — 본체 큐브 바닥(외피로 가려짐), 외피 안쪽 면

## 8. 주의 사항

1. **외피의 마름모는 양식의 핵심** — 단순 큐브로 끝내면 식별성 X
2. **본체 큐브는 외피 안에 숨어 있음** — 외부 시점에서 안 보이지만, 안쪽 가시부(진입부 등)에서 일부 노출 가능
3. **상단 옥상은 평지** — 광장·정원 있을 수 있으나 외부에서 안 보이므로 모델링 X
4. **텐션 케이블** — 외피를 잡아주는 가는 케이블이 있지만 폴리곤 비용 큼 → 디테일 단계에서 결정
